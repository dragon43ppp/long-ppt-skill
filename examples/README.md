# Examples

这里放 `Long-ppt skill` 的小型说明和路线映射。

当前仓库更偏真实案例，优先看 `cases/`。

## 1. `long-ppt-native`

适合：

- 架构图
- 框架图
- 附件页

可参考：

- `cases/power-observability-native/电力可观测性架构图_可编辑重建版_v1.pptx`

## 2. `long-ppt-rebuild`

适合：

- 单页图片稿
- 截图页
- 图片版 PPTX

可参考：

- `cases/ppt2-rebuild-demo/arch_like_original_editable.pptx`
- `cases/ppt2-rebuild-demo/unified_ops_arch_gpt_probe.png`

## 3. `long-ppt-image`

适合：

- 封面
- 背景
- 结构示意图和抽象视觉页

可参考：

- `cases/ai-architecture-to-editable-ppt.md`

## 4. `long-ppt-core`

适合：

- PPT 与 Word 材料总结
- 大纲重写
- 整套 PPT 路线编排

## 5. `long-ppt-hybrid`

适合：

- 截图保真页
- 大屏页
- 仪表盘页

## 6. `long-ppt-vba`

适合：

- Office 宏环境
- PowerPoint 内自动执行重建

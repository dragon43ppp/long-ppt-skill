# Examples

这个目录用于放 `Long-ppt skill` 的最小可运行示例说明。

当前仓库已经附带的真实案例主要放在 `cases/` 下。

如果你想快速理解每条路线，可以先看下面这个映射：

## 1. `long-ppt-native`

适合：

- 架构图
- 框架图
- 附件页

可参考：

- `cases/power-observability-native/电力可观测性架构图_可编辑重建版_v1.pptx`

## 2. `long-ppt-rebuild`

适合：

- 现有图片页
- 截图页
- 图片版 PPTX

可参考：

- `cases/ppt2-rebuild-demo/arch_like_original_editable.pptx`
- `cases/ppt2-rebuild-demo/unified_ops_arch_gpt_probe.png`

## 3. `long-ppt-image`

适合：

- 封面
- 背景
- 替代真实项目图的示意图

可参考：

- `cases/ai-architecture-to-editable-ppt.md`

## 4. `long-ppt-core`

适合：

- PPT + Word 材料总结
- 大纲重写
- 整套 PPT 路线编排

## 5. `long-ppt-hybrid`

适合：

- 截图型监控页
- 大屏页
- 仪表盘页

## 6. `long-ppt-vba`

适合：

- Office 宏环境
- PowerPoint 端自动执行重建

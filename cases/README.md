## 可以怎么理解这些案例

- 现有 PPT + Word 材料重写，属于 `long-ppt-core`
- 截图页、监控页、大屏页重建，属于 `long-ppt-hybrid`
- 按原版式把一页视觉稿变成可编辑 PPT，属于 `long-ppt-rebuild`
- 架构图、附件页、规则清晰的结构页重建，属于 `long-ppt-native`

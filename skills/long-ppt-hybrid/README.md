# long-ppt-hybrid

`long-ppt-hybrid` 是 `Long-ppt skill` 里专门处理“复杂视觉页面”的可编辑重建路线。

它的核心不是全重画，而是做一个更实用的平衡：

- 复杂视觉区域尽量保留
- 真正需要修改的文字和结构改成可编辑对象

## 最适合它的页面

- 监控页
- 仪表盘页
- 大屏页
- 截图页
- 设计感较强但不值得全原生重画的页面

## 它为什么重要

这类页面如果强行全原生重建，成本会很高，而且容易失真。

Hybrid 的价值就在于：

- 保住原始视觉质感
- 让关键内容能继续编辑
- 把时间花在真正值得编辑的部分

## 典型可编辑部分

- 标题
- 长文案
- 标签
- 说明块
- 表格
- 指标卡片
- 实用注释

## 什么时候不要用它

如果页面本质上是：

- 架构图
- 框架图
- 模块图
- 规则非常明确的结构页

通常应优先考虑 `long-ppt-native`。

## 一句话定位

`long-ppt-hybrid` 是面向截图页、大屏页、监控页和仪表盘页的“保视觉 + 保可编辑性”的混合重建能力。

# Long-ppt Skills Index

## Skill Map

| Skill | Primary role | Best for |
|------|------|------|
| `long-ppt-core` | Structure-first orchestration | deciding the route for a page or full deck |
| `long-ppt-image` | image and backplate generation | covers, backgrounds, structure backplates |
| `long-ppt-hybrid` | hybrid editable reconstruction | screenshot-heavy or dashboard-heavy pages |
| `long-ppt-native` | python-pptx native rebuilding | architecture diagrams, attachment pages, framework slides |
| `long-ppt-vba` | VBA-native rebuilding | macro-driven PowerPoint reconstruction workflows |

## Recommended entry point

If you are not sure where to start:

1. start with `long-ppt-core`
2. decide whether the page needs redesign or reconstruction
3. if redesign is needed, go to `long-ppt-image`
4. if screenshot fidelity matters, go to `long-ppt-hybrid`
5. if native editable diagram rebuilding matters, go to `long-ppt-native`
6. if your workflow is PowerPoint macro-centric, go to `long-ppt-vba`
